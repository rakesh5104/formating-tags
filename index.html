HTML

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web Pages
  • It was introduced in 1991 by Tim Berners Lee

HEADING TAGS

HTML headings are titles or subtitles that you want to display on a webpage. HTML headings are defined with theh1toh6 tags.

  • h1 is the largest heading
  • h6 is the smallest heading

PARAGRAPH TAG

The paragraph in html is used to define a paragraph text.

  • HTML paragraphs are defined with the p tag.

    • BLOCKLINE ELEMENTS

    Block level elements in HTML are elements that typically start on a new line and occupy the full width avalible to them such as paragraph tag.

    INLINE ELEMENTS

    Inline elements in HTML are elements that do not start on a new line and only occupy as much as necessary, allowing elements to sit beside them horizontally.

    TABLE TAGS

    • It is used to represent content in a tabular form
    • It is defines with table tag
    • Rows are represented with tr
    • Columns are represented with td
    • Columns headings are represented with th

    TABLE FEATURES

    1. Table caption:It is used provide title for a table
    2. Cell padding: It is used to create space around content in a cell
    3. Cell spacing: It is used to create a space between cells
    4. Width and Height: It is used to define height and width to an element

    IMAGE TAG

    The image tag in HTML is used to embed images into a webpage. It is an empty tag that requires the src attribute to specify the image source URL, and typically includes the alt attribute to provide alternative text if the image cannot be displayed

    ANCHOR TAG

    The anchor tag is used to create hperlinks, allowing users to navigate to other webpages, files, or locations within the same page.
